How to merge cells horizontally in GridView via the CustomDrawCell event

This example illustrates an approach that is similar to that one described in the following thread: http://www.devexpress.com/scid=E2472. The mentioned functionality is implemented via the GridView.CustomDrawCell event handling. In addition, this example illustrates how to implement merged text scrolling.
